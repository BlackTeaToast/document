# 上架整合文档

## Listing Introduction
Listing 是为服务上架到 EnSaaS 4.0 Catalog 和 Marketplace 提供上架的方案和 api.


## 上架整合流程

- 上架服务流程  
```bash
 上架服务 ---> 上架服务对应的方案 ---> 上架服务方案料号
```
 [上架服务](#service) ---> [上架服务对应的方案](#servicePlan) ---> [上架服务方案料号](#pn)
