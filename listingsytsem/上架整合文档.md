# 上架整合文档

## 上架整合流程
### 上架服务流程

```bash
 上架服务 ---> 上架服务对应的方案 ---> 上架服务方案料号
```
### 上架服务步骤
1. 上架服务，调用 [上架服务接口](#上架服务接口) 进行上架
2. 上架服务对应的方案，调用 [上架服务方案接口](#上架服务方案接口) 进行上架
3. 上架服务方案料号，调用 [上架服务料号接口](#上架服务料号接口) 进行上架

### 上架服务包流程

```bash
 上架包服务 ---> 上架服务方案料号
```
### 上架服务包步骤
1. 上架服务包，调用 [上架服务包接口](#上架服务包接口) 进行上架
2. 上架服务方案料号，调用 [上架服务料号接口](#上架服务料号接口) 进行上架
